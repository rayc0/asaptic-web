电动汽车充电连接器兼容性——各市场GB/T 20234、IEC 62196、SAE J1772与NACS标准解析

全面概述中国、美国、欧盟、英国和澳大利亚电动汽车充电连接器标准，涵盖GB/T 20234、IEC 62196 Type 2/CCS2、SAE J1772、CCS1与NACS，并比较交流和直流充电模式、协议兼容性及出口充电设备的互操作性考量。

中国：GB/T 20234——国内标准基准

中国电动汽车充电连接器领域由GB/T 20234系列标准主导，该系列为国家推荐性标准（GB/T，即自愿性，但在中国国内市场销售和车辆销售中事实上具有强制性），由国家标准化管理委员会（SAC）发布。该系列现有三个有效部分：

  • GB/T 20234.1:2015——电动汽车传导充电用连接装置通用要求
  • GB/T 20234.2:2015——交流充电接口（模式3充电，单相和三相，最大32 A，采用7针连接器）
  • GB/T 20234.3:2015——直流充电接口（物理形态类似CHAdeMO，但协议不同；额定最大250 A、1000 V直流）

GB/T 20234.2交流连接器采用7针格式，外形独特，与IEC 62196 Type 2（欧洲交流连接器）、SAE J1772（北美交流连接器）或其他国际标准连接器不能直接互换。GB/T 20234.3直流连接器同样采用专有物理形态，与CCS1、CCS2或CHAdeMO不同。

对于从中国出口电动汽车充电器至西方市场的制造商，GB/T连接器设计不能简单地重新贴标用于目标市场。需要进行硬件改造——安装符合目标市场标准的不同连接器头。GB/T 20234.3底层通信协议（直流充电控制采用GB/T 27930）同样有别于IEC 61851-1（通用电动汽车充电）和DIN SPEC 70121 / ISO 15118（CCS充电桩使用的协议层）。

中国已参与新一代连接器标准（有时称为超充/HPC，与日本CHAdeMO协会联合开发）的国际标准化讨论，旨在实现中国及潜在其他市场的兼容。截至知识截止日期，GB/T 20234.3仍是已部署基础设施的现行中国直流充电标准。

IEC 62196与CCS：欧洲及全球交直流标准

IEC 62196——电动汽车传导充电用插头、插座、车辆连接器和车辆插口是电动汽车充电连接器国际标准系列，分为多个部分：

  • IEC 62196-1:2022——通用要求
  • IEC 62196-2:2022——交流插针与插管附件的尺寸兼容性和互换性要求（涵盖Type 1 / SAE J1772和Type 2 / Mennekes连接器几何形状）
  • IEC 62196-3:2022——直流及交直流插针与插管车辆耦合器的尺寸兼容性和互换性要求（涵盖CCS1、CCS2和CHAdeMO几何形状，以及GB/T格式）

Type 2（IEC 62196-2）是欧盟自2014年起依据《替代燃料基础设施指令/法规》（AFIR）对新充电基础设施强制要求的标准交流充电连接器。支持单相和三相交流充电，最大43 kW（三相63 A）。模式3充电（IEC 61851-1定义）是Type 2交流充电的工作模式。

CCS2（联合充电系统，IEC 62196-3）是欧盟、英国和澳大利亚的标准直流快充连接器。它在Type 2交流连接器下方增加两个直流功率引脚，使车辆可通过同一插口进行交直流充电。CCS2通过ISO 15118（车网通信，支持即插即充）或DIN SPEC 70121（仍广泛使用的早期协议）通信。欧盟依据AFIR分阶段强制要求新建直流充电桩支持CCS2。

对于澳大利亚市场，《AS/NZS 3000》（布线规则）和AS 4777新兴并网要求适用于电动汽车充电桩安装。澳大利亚已将IEC 62196 Type 2用于交流充电、CCS2用于直流快充，作为新建公共充电基础设施的事实标准，与欧盟方式一致。

对于英国市场，IEC 62196 Type 2和CCS2同样是现行标准，并有英国充电桩法规的特定要求（包括开放标准协议OCPP 1.6及智能充电能力）。

北美：SAE J1772、CCS1与NACS

自2023年以来，北美电动汽车充电连接器领域经历了重大变革，目前有三种连接器标准与充电设备出口商密切相关。

SAE J1772——SAE电动汽车和插电式混合动力汽车传导充电耦合器是美国、加拿大和日本使用的标准交流充电连接器。Type 1（J1772）连接器支持最大19.2 kW（单相80 A）的模式3交流充电。其物理形态与IEC 62196-2 Type 1连接器相同。SAE J1772还定义了北美交流模式2和模式3充电所用的控制导引通信信号，该信号与IEC 61851-1欧洲框架中的信号相同，但连接器不同。

CCS1（联合充电系统，北美）在J1772连接器下方增加两个直流功率引脚，实现直流快充。CCS1在物理形态上与CCS2（采用Type 2交流连接器底座）不同。CCS1通过DIN SPEC 70121或ISO 15118通信。2023年前，CCS1是北美非特斯拉车辆的主流直流快充标准。

NACS——北美充电标准（SAE J3400）原为专有连接器设计，现已标准化为SAE J3400，并被美国主要汽车制造商采用为车辆侧连接器。美国联邦政府（能源与交通联合办公室）和主要充电网络运营商已承诺支持NACS。截至2024至2025年，在联邦补助条件（NEVI计划）下，美国新建直流快充桩越来越多地被要求支持NACS（原生支持或通过转接器）。NACS支持交流充电（功率水平与J1772相近）和直流快充（计划扩展至最大1 MW）。

对于面向北美市场的电动汽车充电器出口商：交流充电桩须支持J1772（或配备J1772至NACS转接器）；根据现行NEVI计划要求，面向公共部署的新建直流快充桩须同时支持CCS1和NACS；新建联邦资助基础设施中的即插即充功能需支持ISO 15118协议。UL 2594（电动汽车供电设备标准）和UL 2231（电动汽车供电回路人身保护系统）是美国市场认证的主要安全标准。

协议兼容性、互操作性与认证要求

物理连接器兼容性是电动汽车充电器进入市场的必要条件，但并非充分条件。通信协议、认证要求和并网交互标准构成了因市场而异的额外强制性要求层。

通信协议。充电站与车辆之间的控制通信决定了充电会话能否启动、监控和安全终止。主要协议包括：

  • IEC 61851-1 / 模式3：欧盟、英国、澳大利亚和北美交流充电使用的基本控制导引（CP）信号。尽管连接器不同，各市场之间电气兼容。
  • DIN SPEC 70121：CCS1和CCS2充电桩使用的直流充电通信协议（过渡协议）。已广泛部署，但正逐步被ISO 15118取代。
  • ISO 15118：支持即插即充（自动认证）、智能充电和双向（V2G）充电的车网通信协议。在美国（NEVI计划）新建联邦资助直流基础设施中为强制要求；在欧盟用于即插即充为必要条件；在英国和澳大利亚智能充电合规方面的要求日益增多。
  • OCPP（开放充电点协议）：充电点管理系统（CPMS）与充电站之间的后端通信协议。OCPP 1.6和2.0.1为现行版本。英国充电桩法规要求至少支持OCPP 1.6。许多全球网络运营商要求新建设施支持OCPP 2.0.1。
  • GB/T 27930：中国直流充电通信协议。中国以外不使用。采用GB/T 27930的充电桩出口至IEC/SAE市场时，须替换通信协议栈。

出口充电器的认证要求。

  • 欧盟：须通过《低压指令》（LVD 2014/35/EU）和《EMC指令》（2014/30/EU）的CE认证。AFIR要求智能充电能力和OCPP支持。须符合IEC 61851-1和IEC 62196系列。
  • 英国：UKCA或CE标志（过渡期间）。须符合BS EN 61851-1、BS EN 62196系列及《2021年电动汽车（智能充电桩）条例》（要求OCPP 1.6、智能充电、需求侧响应和数据记录）。
  • 美国：商业部署须获得NRTL颁发的UL 2594认证。任何无线/网络组件须符合FCC第15部分规定。联邦资助站点的NEVI计划要求支持NACS和ISO 15118。
  • 澳大利亚：须通过电气设备安全体系（EESS）下的RCM（合规监管标志）。须符合AS/NZS 3000、AS 4777（并网）及已作为AS标准采用的IEC 61851/62196相关标准。并网系统须获得CEC批准。
