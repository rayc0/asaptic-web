電動汽車充電連接器兼容性——各市場GB/T 20234、IEC 62196、SAE J1772與NACS標準解析

全面概述中國、美國、歐盟、英國和澳大利亞電動汽車充電連接器標準，涵蓋GB/T 20234、IEC 62196 Type 2/CCS2、SAE J1772、CCS1與NACS，並比較交流和直流充電模式、協議兼容性及出口充電設備的互操作性考量。

中國：GB/T 20234——國內標準基準

中國電動汽車充電連接器領域由GB/T 20234系列標準主導，該系列為國家推薦性標準（GB/T，即自願性，但在中國國內市場銷售和車輛銷售中事實上具有強制性），由國家標準化管理委員會（SAC）發布。該系列現有三個有效部分：

  • GB/T 20234.1:2015——電動汽車傳導充電用連接裝置通用要求
  • GB/T 20234.2:2015——交流充電接口（模式3充電，單相和三相，最大32 A，採用7針連接器）
  • GB/T 20234.3:2015——直流充電接口（物理形態類似CHAdeMO，但協議不同；額定最大250 A、1000 V直流）

GB/T 20234.2交流連接器採用7針格式，外形獨特，與IEC 62196 Type 2（歐洲交流連接器）、SAE J1772（北美交流連接器）或其他國際標準連接器不能直接互換。GB/T 20234.3直流連接器同樣採用專有物理形態，與CCS1、CCS2或CHAdeMO不同。

對於從中國出口電動汽車充電器至西方市場的製造商，GB/T連接器設計不能簡單地重新貼標用於目標市場。需要進行硬件改造——安裝符合目標市場標準的不同連接器頭。GB/T 20234.3底層通信協議（直流充電控制採用GB/T 27930）同樣有別於IEC 61851-1（通用電動汽車充電）和DIN SPEC 70121 / ISO 15118（CCS充電樁使用的協議層）。

中國已參與新一代連接器標準（有時稱為超充/HPC，與日本CHAdeMO協會聯合開發）的國際標準化討論，旨在實現中國及潛在其他市場的兼容。截至知識截止日期，GB/T 20234.3仍是已部署基礎設施的現行中國直流充電標準。

IEC 62196與CCS：歐洲及全球交直流標準

IEC 62196——電動汽車傳導充電用插頭、插座、車輛連接器和車輛插口是電動汽車充電連接器國際標準系列，分為多個部分：

  • IEC 62196-1:2022——通用要求
  • IEC 62196-2:2022——交流插針與插管附件的尺寸兼容性和互換性要求（涵蓋Type 1 / SAE J1772和Type 2 / Mennekes連接器幾何形狀）
  • IEC 62196-3:2022——直流及交直流插針與插管車輛耦合器的尺寸兼容性和互換性要求（涵蓋CCS1、CCS2和CHAdeMO幾何形狀，以及GB/T格式）

Type 2（IEC 62196-2）是歐盟自2014年起依據《替代燃料基礎設施指令/法規》（AFIR）對新充電基礎設施強制要求的標準交流充電連接器。支持單相和三相交流充電，最大43 kW（三相63 A）。模式3充電（IEC 61851-1定義）是Type 2交流充電的工作模式。

CCS2（聯合充電系統，IEC 62196-3）是歐盟、英國和澳大利亞的標準直流快充連接器。它在Type 2交流連接器下方增加兩個直流功率引腳，使車輛可透過同一插口進行交直流充電。CCS2透過ISO 15118（車網通信，支持即插即充）或DIN SPEC 70121（仍廣泛使用的早期協議）通信。歐盟依據AFIR分階段強制要求新建直流充電樁支持CCS2。

對於澳大利亞市場，《AS/NZS 3000》（佈線規則）和AS 4777新興並網要求適用於電動汽車充電樁安裝。澳大利亞已將IEC 62196 Type 2用於交流充電、CCS2用於直流快充，作為新建公共充電基礎設施的事實標準，與歐盟方式一致。

對於英國市場，IEC 62196 Type 2和CCS2同樣是現行標準，並有英國充電樁法規的特定要求（包括開放標準協議OCPP 1.6及智慧充電能力）。

北美：SAE J1772、CCS1與NACS

自2023年以來，北美電動汽車充電連接器領域經歷了重大變革，目前有三種連接器標準與充電設備出口商密切相關。

SAE J1772——SAE電動汽車和插電式混合動力汽車傳導充電耦合器是美國、加拿大和日本使用的標準交流充電連接器。Type 1（J1772）連接器支持最大19.2 kW（單相80 A）的模式3交流充電。其物理形態與IEC 62196-2 Type 1連接器相同。SAE J1772還定義了北美交流模式2和模式3充電所用的控制導引通信信號，該信號與IEC 61851-1歐洲框架中的信號相同，但連接器不同。

CCS1（聯合充電系統，北美）在J1772連接器下方增加兩個直流功率引腳，實現直流快充。CCS1在物理形態上與CCS2（採用Type 2交流連接器底座）不同。CCS1透過DIN SPEC 70121或ISO 15118通信。2023年前，CCS1是北美非特斯拉車輛的主流直流快充標準。

NACS——北美充電標準（SAE J3400）原為專有連接器設計，現已標準化為SAE J3400，並被美國主要汽車製造商採用為車輛側連接器。美國聯邦政府（能源與交通聯合辦公室）和主要充電網絡運營商已承諾支持NACS。截至2024至2025年，在聯邦補助條件（NEVI計畫）下，美國新建直流快充樁越來越多地被要求支持NACS（原生支持或透過轉接器）。NACS支持交流充電（功率水平與J1772相近）和直流快充（計畫擴展至最大1 MW）。

對於面向北美市場的電動汽車充電器出口商：交流充電樁須支持J1772（或配備J1772至NACS轉接器）；根據現行NEVI計畫要求，面向公共部署的新建直流快充樁須同時支持CCS1和NACS；新建聯邦資助基礎設施中的即插即充功能需支持ISO 15118協議。UL 2594（電動汽車供電設備標準）和UL 2231（電動汽車供電回路人身保護系統）是美國市場認證的主要安全標準。

協議兼容性、互操作性與認證要求

物理連接器兼容性是電動汽車充電器進入市場的必要條件，但並非充分條件。通信協議、認證要求和並網交互標準構成了因市場而異的額外強制性要求層。

通信協議。充電站與車輛之間的控制通信決定了充電會話能否啟動、監控和安全終止。主要協議包括：

  • IEC 61851-1 / 模式3：歐盟、英國、澳大利亞和北美交流充電使用的基本控制導引（CP）信號。儘管連接器不同，各市場之間電氣兼容。
  • DIN SPEC 70121：CCS1和CCS2充電樁使用的直流充電通信協議（過渡協議）。已廣泛部署，但正逐步被ISO 15118取代。
  • ISO 15118：支持即插即充（自動認證）、智慧充電和雙向（V2G）充電的車網通信協議。在美國（NEVI計畫）新建聯邦資助直流基礎設施中為強制要求；在歐盟用於即插即充為必要條件；在英國和澳大利亞智慧充電合規方面的要求日益增多。
  • OCPP（開放充電點協議）：充電點管理系統（CPMS）與充電站之間的後端通信協議。OCPP 1.6和2.0.1為現行版本。英國充電樁法規要求至少支持OCPP 1.6。許多全球網絡運營商要求新建設施支持OCPP 2.0.1。
  • GB/T 27930：中國直流充電通信協議。中國以外不使用。採用GB/T 27930的充電樁出口至IEC/SAE市場時，須替換通信協議棧。

出口充電器的認證要求。

  • 歐盟：須透過《低電壓指令》（LVD 2014/35/EU）和《EMC指令》（2014/30/EU）的CE認證。AFIR要求智慧充電能力和OCPP支持。須符合IEC 61851-1和IEC 62196系列。
  • 英國：UKCA或CE標誌（過渡期間）。須符合BS EN 61851-1、BS EN 62196系列及《2021年電動汽車（智慧充電樁）條例》（要求OCPP 1.6、智慧充電、需求側響應和數據記錄）。
  • 美國：商業部署須獲得NRTL頒發的UL 2594認證。任何無線/網絡組件須符合FCC第15部分規定。聯邦資助站點的NEVI計畫要求支持NACS和ISO 15118。
  • 澳大利亞：須透過電氣設備安全體系（EESS）下的RCM（合規監管標誌）。須符合AS/NZS 3000、AS 4777（並網）及已作為AS標準採用的IEC 61851/62196相關標準。並網系統須獲得CEC批准。
